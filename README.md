# Heavens Bake

  A premium artisan bakery website built with React, Vite, and Tailwind CSS.

  **Live site:** https://daviddchucks-hash.github.io/Heavens-bake/

  ## Project Structure

  ```
  /
  ├── index.html          # Entry point
  ├── css/
  │   └── style.css       # Compiled styles (Tailwind CSS)
  ├── js/
  │   └── script.js       # Compiled React app bundle
  ├── images/             # Product & bakery photography
  ├── assets/             # Site assets (favicon, OG image)
  ├── favicon.svg         # Site icon
  ├── robots.txt          # Search engine directives
  └── .nojekyll           # Disables GitHub Pages Jekyll processing
  ```

  ## Source Code

  The source code lives on the `main` branch and is built with Vite.

  - **Framework:** React 19 + TypeScript
  - **Styling:** Tailwind CSS v4
  - **Animations:** Framer Motion
  - **Routing:** Wouter
